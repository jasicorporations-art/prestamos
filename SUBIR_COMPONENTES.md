# 📤 Subir los Componentes Button e Input a GitHub

He creado los componentes faltantes. Ahora necesitas subirlos a GitHub.

## Comandos en Git Bash

```bash
cd /c/Users/Owner/.cursor
git add components/Button.tsx components/Input.tsx
git commit -m "Agregar componentes Button e Input"
git push
```

O simplemente:

```bash
cd /c/Users/Owner/.cursor
git add .
git commit -m "Agregar componentes Button e Input faltantes"
git push
```

Después del push, Vercel debería hacer un nuevo deployment automáticamente.
