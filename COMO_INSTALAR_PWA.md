# Cómo Instalar la PWA COUNT LEDGER

## ✅ El dominio está configurado correctamente

El dominio **`https://cursor-nu-black.vercel.app`** está activo y funcionando.

## 📱 Pasos para Instalar la PWA

### 1. **Desinstalar la PWA anterior (si existe)**

Si ya tienes la app instalada desde otro dominio, debes desinstalarla primero:

**En Chrome/Edge (Android):**
- Ve a Configuración → Aplicaciones → COUNT LEDGER → Desinstalar

**En Chrome/Edge (Desktop):**
- Ve a `chrome://apps` o `edge://apps`
- Haz clic derecho en COUNT LEDGER → Eliminar

**En iOS Safari:**
- Mantén presionado el icono de la app → Eliminar app

### 2. **Limpiar el caché del navegador**

**En Chrome/Edge:**
- Presiona `Ctrl + Shift + Delete` (Windows) o `Cmd + Shift + Delete` (Mac)
- Selecciona "Imágenes y archivos en caché"
- Haz clic en "Borrar datos"

**En Safari (iOS):**
- Configuración → Safari → Borrar historial y datos del sitio web

### 3. **Acceder al dominio correcto**

**IMPORTANTE:** Debes acceder directamente a:
```
https://cursor-nu-black.vercel.app
```

**NO uses** el dominio temporal como:
- `cursor-hh95w3gir-johns-projects-9d4c1d75.vercel.app`
- `cursor-k88imp9so-johns-projects-9d4c1d75.vercel.app`

### 4. **Instalar la PWA**

**En Chrome/Edge (Android):**
1. Abre `https://cursor-nu-black.vercel.app` en Chrome
2. Verás un banner que dice "Instalar COUNT LEDGER" o un icono de "+" en la barra de direcciones
3. Toca "Instalar" o el icono "+"

**En Chrome/Edge (Desktop):**
1. Abre `https://cursor-nu-black.vercel.app` en Chrome/Edge
2. Busca el icono de instalación en la barra de direcciones (a la derecha)
3. Haz clic en "Instalar"

**En Safari (iOS):**
1. Abre `https://cursor-nu-black.vercel.app` en Safari
2. Toca el botón "Compartir" (cuadrado con flecha)
3. Selecciona "Añadir a pantalla de inicio"
4. Personaliza el nombre si quieres y toca "Añadir"

**En Firefox (Android):**
1. Abre `https://cursor-nu-black.vercel.app` en Firefox
2. Toca el menú (3 puntos) → "Instalar"

### 5. **Verificar la instalación**

Después de instalar:
- Deberías ver el icono "LEDGER ICONO" en tu pantalla de inicio
- El nombre de la app debería ser "COUNT LEDGER"
- Al abrirla, debería funcionar como una app nativa (sin barra de navegador)

## 🔍 Solución de Problemas

### Si el icono no aparece:
1. Asegúrate de haber desinstalado la versión anterior
2. Limpia el caché del navegador
3. Cierra completamente el navegador y vuelve a abrirlo
4. Accede directamente a `https://cursor-nu-black.vercel.app`

### Si el dominio no carga:
1. Verifica tu conexión a internet
2. Intenta en modo incógnito/privado
3. Verifica que estés usando el dominio correcto: `cursor-nu-black.vercel.app`

### Si la app se queda en blanco después de instalar:
1. Desinstala la app
2. Limpia el caché del navegador
3. Vuelve a instalar desde `https://cursor-nu-black.vercel.app`

## 📞 Verificación del Dominio

Para verificar que el dominio está funcionando, puedes:
1. Abrir `https://cursor-nu-black.vercel.app` en tu navegador
2. Deberías ver la página de login de COUNT LEDGER
3. El manifest debería estar disponible en: `https://cursor-nu-black.vercel.app/manifest.json`
4. Los iconos deberían estar en:
   - `https://cursor-nu-black.vercel.app/icons/icon-192x192.png`
   - `https://cursor-nu-black.vercel.app/icons/icon-512x512.png`
