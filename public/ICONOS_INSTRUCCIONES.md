# Instrucciones para Crear los Iconos de la PWA

Para que la PWA funcione correctamente, necesitas crear dos iconos:

## Tamaños Requeridos

1. **icon-192x192.png** - 192x192 píxeles
2. **icon-512x512.png** - 512x512 píxeles

## Opciones para Crear los Iconos

### Opción 1: Usar una Herramienta Online

1. Visitar [PWA Asset Generator](https://github.com/onderceylan/pwa-asset-generator)
2. Subir una imagen de alta resolución (mínimo 512x512)
3. La herramienta generará todos los tamaños necesarios
4. Descargar y colocar los archivos en la carpeta `public/`

### Opción 2: Crear Manualmente

1. Crear o editar una imagen cuadrada de alta resolución
2. Redimensionar a 192x192 y guardar como `icon-192x192.png`
3. Redimensionar a 512x512 y guardar como `icon-512x512.png`
4. Colocar ambos archivos en la carpeta `public/`

### Opción 3: Usar un Generador de Iconos

- [Favicon.io](https://favicon.io/)
- [RealFaviconGenerator](https://realfavicongenerator.net/)
- [PWA Builder](https://www.pwabuilder.com/imageGenerator)

## Recomendaciones

- Usar un logo o imagen representativa de "Inversiones Nazaret Reynoso"
- Asegurarse de que el icono sea reconocible incluso en tamaños pequeños
- Usar colores que contrasten bien
- El icono debe ser cuadrado o tener un fondo cuadrado

## Ubicación de los Archivos

Los iconos deben estar en:
```
public/icon-192x192.png
public/icon-512x512.png
```

Una vez creados, la PWA estará lista para instalarse en dispositivos móviles.
