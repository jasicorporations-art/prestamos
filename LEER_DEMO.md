# 🎨 Demo HTML - Inversiones Nazaret Reynoso

## ✅ Archivo Creado

He creado el archivo **`demo.html`** que contiene una versión completa y funcional de la interfaz de la aplicación.

## 🚀 Cómo Ver la Demo

### Opción 1: Abrir Directamente

1. **Navega a la carpeta del proyecto** en el Explorador de Archivos de Windows
2. **Haz doble clic** en el archivo `demo.html`
3. Se abrirá en tu navegador predeterminado

### Opción 2: Desde el Navegador

1. Abre tu navegador (Chrome, Edge, Firefox, etc.)
2. Presiona `Ctrl + O` (o ve a Archivo > Abrir)
3. Selecciona el archivo `demo.html`
4. ¡Listo!

## 🎯 Lo que Puedes Ver

La demo incluye:

### ✅ Dashboard
- Métricas: Motores disponibles, Total por cobrar, Ventas del mes
- Tabla de ventas recientes

### ✅ Gestión de Motores
- Tabla con motores de ejemplo
- Botón para agregar nuevo motor
- Estados: Disponible/Vendido
- Acciones: Editar/Eliminar

### ✅ Gestión de Clientes
- Tabla con clientes de ejemplo
- Información completa: Nombre, Cédula, Dirección, Garante
- Botón para agregar nuevo cliente

### ✅ Gestión de Ventas
- Lista de ventas realizadas
- Información de cliente, motor, monto, cuotas, saldo
- Botón para ver factura

### ✅ Gestión de Pagos
- Registro de pagos realizados
- Información de cliente, motor, monto, cuota, fecha

### ✅ Factura
- Vista completa de factura
- Optimizada para impresión
- Incluye todos los detalles

## 🎨 Características

- ✅ **Diseño moderno** con Tailwind CSS (CDN)
- ✅ **Responsive** - se adapta a diferentes tamaños de pantalla
- ✅ **Navegación funcional** entre secciones
- ✅ **Modales** para formularios
- ✅ **Estilos de impresión** para facturas
- ✅ **Interfaz limpia y profesional**

## 📝 Notas

- Los datos son **simulados** (no se guardan realmente)
- Los formularios muestran **alertas** al guardar (funcionalidad simulada)
- La interfaz es **idéntica** a la versión Next.js
- Puedes ver cómo se ve y funciona sin necesidad de configurar nada
